caçador de peixe é jogos de bónus com excelentes gráficos 3D para várias características oceânicas, atraente para qualquer jogador para o jogo captura rentável jogo peixes. Na pesca do jogo de vídeo pode ir cabeça-a-cabeça em uma corrida contra o outro para pegar tantos peixes e mais rentável possível, marcar pontos para obter o maior pagamento! Este conceito simples é compensado pelo rápido jogo, emocionante que exige raciocínio rápido, habilidade e coordenação, a fim de ser o melhor!

Como jogar:
- Moedas Inserir para iniciar jogos;
- Assim que o jogo começou a tocar um alvo para atirar e pegar o peixe;
- Jogador também pode usar o botão de mudança para mudar entre as 9 armas a qualquer momento durante o jogo, que têm vários tamanhos líquidos e energia níveis jogador está usando;
- Use itens de power-up para melhorar o seu jogo
- Tente completar todas as missões
